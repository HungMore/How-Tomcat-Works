# How-Tomcat-Works

本仓库的代码源于《How Tomcat Works》（《深入剖析Tomcat》）

## Why
书中的代码年代较为久远，不能直接在当前的环境下运行。  

因此我稍微修改了一部分由于JDK版本变化引起的Bug（enum关键字），同时通过Maven进行依赖管理。  

目的只有一个：在JDK8和Maven3.6的环境下可以运行。

## Use
环境：JDK8 & Maven3.6  

建议:使用IDEA更好，可以方便查看依赖包的源码。  

*每一章的单个小程序都是独立运行的*  

*每一章的程序Run起来之后，测试的URL和书中一致*
